# Affiliated Marketing

A simple web project for affiliated marketing content.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `pics/` - Directory for images

## Getting Started

1. Open `index.html` in your web browser to view the site
2. Customize the content in `index.html` and styles in `style.css`
3. Add images to the `pics/` directory

## Deployment

This project is ready to be deployed on Vercel, GitHub Pages, or any static hosting service.
